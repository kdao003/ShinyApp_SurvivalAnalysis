# ShinyApp_SurvivalAnalysis

Shiny App that produces a survival curve and cox proportional hazards model for survival data. Users can upload a google sheet version of the dataset with a provided google sheets link.

The dataset used to create this app was BEACON Moffitt (http://beacon.moffitt.org/BEACON_Directory/).

Browser Version of the App found here: https://kdao144.shinyapps.io/ShinyApp_LongitudinalAnalysis/
